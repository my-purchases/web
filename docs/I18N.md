## Adding a New Language

1. Create `src/i18n/locales/{code}.json` (copy `en.json` as a template)
2. Translate all keys
3. Register the import in `src/i18n/index.ts`:

```typescript
import newLang from './locales/xx.json';

// Add to resources:
resources: {
  // ...existing
  xx: { translation: newLang },
}
```

4. Add the language name to all locale files under the `language` section:
```json
"language": {
  "xx": "Language Name"
}
```

5. If RTL, add the code to the `rtlLanguages` array in `src/i18n/index.ts`

## Translation Keys

Translations are organized by section:

- `common` — Shared UI labels (Save, Cancel, Close, etc.)
- `nav` — Navigation items
- `invitation` — Welcome screen
- `providers` — Provider panel (names, actions, coming-soon label)
- `purchases` — Purchase listing, selection actions, cost calculator
- `tags` — Tag system
- `groups` — Purchase groups
- `exportImport` — Data export/import
- `settings` — Settings page
- `about` — About page
- `privacy` — Privacy policy
- `currency` — Currency setup and conversion
- `language` — Language names (in native script)

## RTL Support

Arabic is automatically detected as RTL. The `dir` attribute on `<html>` is updated when switching to/from RTL languages. Tailwind CSS handles layout mirroring.
